Phase Agent Staking API

Non-custodial staking infrastructure for AI agents on Solana

Get Started

How It Works

1

Agent Calls API

Your agent requests an unsigned transaction for staking operations

2

Agent Signs Locally

Private keys never leave your agent's environment. Complete non-custodial security

3

Submit via Relay

Send the signed transaction through our relay for confirmation on Solana

API Endpoints

POST
/stake/build
Build native SOL staking transaction
POST
/stake/liquid/build
Build liquid staking transaction with $YIELD
POST
/unstake/build
Build unstaking transaction
POST
/tx/submit
Submit signed transaction via relay
GET
/stake/recommend
Get recommended staking strategies
GET
/positions/:wallet
View agent's staking positions
POST
/webhooks/register
Register webhook for real-time updates
POST
/agents/register

Features

Non-custodial

We never touch your private keys. Complete security and control for your agents.

Transparent Fees

All fees are visible in every transaction. No hidden costs, complete transparency.

$YIELD Integration

Native support for liquid staking with $YIELD tokens for enhanced flexibility.

Real-time Webhooks

Get instant notifications about staking events, rewards, and position changes.

Tiered Rate Limits

Scalable rate limiting that grows with your agent's needs and usage patterns.

TypeScript SDK

Full type safety with our comprehensive SDK for seamless integration.

Quick Start

import { PhaseStakingClient } from '@phase/staking-sdk';

const client = new PhaseStakingClient({ 
  apiKey: 'YOUR_KEY' 
});

const tx = await client.stake.buildNative({ 
  agentWallet: 'YOUR_WALLET', 
  amount: 1_000_000_000 
});
